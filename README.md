# anti-jam-encode

Simple calculation of anti-jamming methods.

## Usage 

First of all you must clone this project and then cd to the project folder.

### Rust

1. Cd to the "rust/hash-trash" dir;
2. To build:
```
$ cargo build --release
```
3. To run:
```
$ cargo run --release
```

### Python

1. Cd to the "python" dir;
2. Then run:
```
$ python main.py
```
